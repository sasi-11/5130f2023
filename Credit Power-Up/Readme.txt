PROJECT TITLE : Credit Power-Up

ABSTRACT:  The E-Coerce Credit Vitality Initiative represents a novel approach to addressing financial inclusion and credit access for underserved populations, particularly in the digital era. This innovative program leverages digital technologies, data analytics, and financial literacy to empower individuals and businesses to build and enhance their creditworthiness while fostering responsible financial behavior.
 
INTERFACE: In this project, we've incorporated a variety of interfaces, including SignUp, SignIn for user authentication, a Home Page for navigation,and a Signout interface for user management.

DATASETS:   

We Will be using Different API's/ Services for our project.

a. Figma:User interface and user experience design, with an emphasis on real-time collaboration,utilising a variety of vector graphics editor and prototyping tools.
b. OPEN API's :  The OpenAI API can also be used to generate and edit images or convert speech into text.
c. Google Firebase: For Authentication and Authorization.
d. AWS cloud storage: Used to store and organize the data.

Technology Stack : Javascript, React, Redux, HTML, Figma, Aws.
